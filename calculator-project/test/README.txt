Information on the different packages found in the /testing directory:

- acceptance -
Contains test cases to show the client requirements which the current system satisfies.

- regression - 
Contains a subset of the test cases from 'unit' which verify there are no uninteded effects on the main functionality of the system, after modifications have been made.

- system - 
Contains both system and functional test cases to evaluate the program's compliance with client requirements.

- unit - 
Contains both unit and integration tests used during the development of the program to verify that code units work as intended, as well as units working together.
